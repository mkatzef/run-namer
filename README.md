# Run Namer
Generates names for run activity logs

## Usage
For a demo, run the python3 script `run-namer` without args:
`./run-namer.py`

To name a run at a specific time, call the script with a time argument:
`./run-namer.py --time 14:30`
(For 2:30 pm).
